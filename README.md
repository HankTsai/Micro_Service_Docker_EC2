# Micro_Service_Docker_EC2

### 項目介紹：

此repo主要是練習AWS的雲原生系統
主要是以EC2與S3服務為基底，將linebot的功能容器化
RDB的部分因為要另外開銷，我直接將mySQL綁成容器加在ymal檔裡

這隻微服務只保留以下三個linbot的基本運作功能：
1. 食譜推薦
2. 賣菜助手（食材即時辨識）
3. 冰箱倉庫查看

未來有機會可以將IOT的部分也雲端化，交由AWS_IOT_core整合。
